# 🏥 AI Healthcare Insurance Fraud Detection System

An end-to-end Machine Learning project that detects fraudulent healthcare insurance claims using Python, Scikit-learn, and Streamlit. The system analyzes healthcare claim data to identify suspicious patterns, predict fraud risk, and provide actionable insights through an interactive dashboard.

---

## 🚀 Features

- Healthcare claim fraud prediction
- Data preprocessing & feature engineering
- Multiple ML model comparison
- Explainable AI predictions
- Interactive Streamlit dashboard
- Batch CSV prediction
- Provider analytics
- Automatic report generation

---

## 🤖 AI Assistant

The project includes an integrated AI Assistant that enhances the user experience by providing explanations for fraud predictions, interpreting risk levels, answering project-related questions, and assisting users in understanding healthcare insurance claim analysis through an interactive interface.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Ayushkumarpal008/AI-Healthcare-Insurance-Fraud-Detection-System.git
```

Move into the project folder

```bash
cd AI-Healthcare-Insurance-Fraud-Detection-System
```

Install the required dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Train the model

```bash
python -m src.train
```

Run prediction

```bash
python -m src.predict --input data/processed/sample_prediction_input.csv
```

Launch the Streamlit application

```bash
streamlit run app/app.py
```
